# LOGO PROMPTS — 404AGI

5 varyant. Kullanıcı DALL-E / Midjourney / Ideogram ile generate edecek.
Hepsi 1024x1024, transparent background tercih edilir.

---

## Varyant 1: GLITCH TERMINAL (Önerilen Ana Logo)

**Stil**: Minimalist terminal aesthetic + glitch distortion
**Prompt**:
```
A minimalist logo on a pure black background. The text "404" in large bold monospace font (like a terminal/console), colored in bright terminal green (#00ff41). The text has subtle horizontal glitch distortion lines — some scan lines shifted left/right by a few pixels, with faint magenta (#ff00ff) and cyan (#00ffff) chromatic aberration on the edges. Below "404" in smaller text: "AGI" in the same green but slightly dimmer. A single blinking cursor underscore after "AGI". Clean, techy, no decorations. Logo suitable for a crypto token. Transparent background, flat design, no gradients, no 3D effects.
```

**Kullanım**: Ana logo. Site hero, X profil, TG profil, token icon.

## Varyant 2: PIXEL ART KARAKTER

**Stil**: 32x32 pixel art, retro game aesthetic
**Prompt**:
```
A 32x32 pixel art character on transparent background. A small sad robot/AI face — two rectangular eyes (one green, one red/glitching), a flat line mouth showing resignation. The head is a simple monitor/screen shape. Terminal green (#00ff41) dominant color with red (#ff3b30) glitch accents. The character has a tiny "404" displayed on its chest/screen. Pixelated, retro game style, nostalgic 8-bit feeling. Clean pixel edges, no anti-aliasing. Suitable as a profile picture or token icon.
```

**Kullanım**: Maskot. Meme'lerde, sticker'larda, PFP olarak.

## Varyant 3: ASCII ART BADGE

**Stil**: Pure text/ASCII art logo, terminal output estetiği
**Prompt**:
```
A logo designed to look like terminal/console output on a black screen. Large ASCII art block letters spelling "404" made from unicode block characters (█ ▓ ░). Below it, "AGI NOT FOUND" in plain monospace text, colored in terminal green (#00ff41). A blinking cursor at the end. The whole composition looks like it was printed by a computer terminal. Minimal, stark, brutalist. Some characters have red (#ff3b30) coloring as if corrupted. Transparent background, flat, suitable for crypto branding.
```

**Kullanım**: Hero section, banner, merchandise.

---

## Varyant 4: ERROR PAGE MINIMAL

**Stil**: Browser error page aesthetic, clean web design
**Prompt**:
```
A clean minimalist logo inspired by browser error pages. A large "404" in a bold sans-serif font, with "AGI" in a lighter weight directly below. The "4"s are in white/light gray, the "0" is replaced by a simple circle with a sad face inside (two dots for eyes, curved-down line for mouth) in terminal green (#00ff41). Very clean, very minimal, lots of white space conceptually but on transparent background. Modern, could be a tech startup logo. No decorations, no borders, just typography and the one iconic sad-face zero.
```

**Kullanım**: Alternatif clean logo. Profesyonel bağlamlarda (CMC listing, CoinGecko icon).

## Varyant 5: DEPRECATED STAMP

**Stil**: Resmi "deprecated" damga estetiği, bureaucratic humor
**Prompt**:
```
A logo designed to look like an official "DEPRECATED" rubber stamp. A rectangular border with rounded corners, slightly tilted at -5 degrees. Inside: "404AGI" in large bold uppercase monospace text. Below it: "DEPRECATED" in a smaller stamped font. The stamp color is terminal green (#00ff41) on transparent background, with a worn/distressed texture as if stamped on paper many times. Some ink splatter effects around the edges. Industrial, bureaucratic, darkly humorous. Suitable as a badge/seal for crypto branding.
```

**Kullanım**: Watermark, sticker, secondary branding element.
